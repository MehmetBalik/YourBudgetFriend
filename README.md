# YourBudgetFriend
YourBudgetFriend is a smart tool that automatically visits online stores and collects information about products you're interested in. It helps you keep track of prices, stock availability, and other details, making it easier to manage your shopping and budget

## Setup and Running Instructions

To get started with YourBudgetFriend, follow these steps:

1.  **Install Dependencies:**
    Ensure you have Python and pip installed. Then, install the required libraries using pip:
    
    pip install -r requirements.txt

2.  **Run the Server:**
    Navigate to the project's root directory in your terminal and start the Flask server:

    python server.py
    
    This will start the backend server and serve the user interface. You can then access the application through your web browser.
